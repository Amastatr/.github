# AI Contextualizers

Small Chrome extensions by Amastatr Innovation Haus that add the context AI chat interfaces leave out. TimeStamp adds when, GeoStamp adds where, UserStamp adds who. Not a platform, not a fix for your workflow, just standing context supplied on every message.

One discipline runs through all three: stamps fire once, on send only, and everything stays local. Manifest V3, no accounts, no telemetry, nothing collected in the background.

## The extensions

[TimeStamp](https://github.com/Amastatr/TimeStamp) adds the real date, time, and timezone to every message you send.

[GeoStamp](https://github.com/Amastatr/GeoStamp) adds your location at the precision you choose, off by default.

[UserStamp](https://github.com/Amastatr/UserStamp) tags each message with the team member sending it, for shared subscriptions.

All three run on ChatGPT, Claude, Gemini, and Grok, and load unpacked from their repositories.
